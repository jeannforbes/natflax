Once you have installed the database and populated it, here's how to use it (there are no passwords on users for convenience):

First run NatFlax.java

Here's how to rent a book:
1: Select Customer: log in as "mdelacourt0"
2: Search for Books: "Fintone"
3: Say "yes"
4: Enter the book's ISBN (on screen): "939482509-6"
5: Select Store with ID "19"
5: Check out your rentals, to confirm it has been rented!

Return the book you just rented:
1: Log in as an employee: ID  "20"
2: Return a book: ISBN "939482509-6"
3: Select which customer has returned the book (CID "1")
